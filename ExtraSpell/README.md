# Extra Spell
Mod Url ———— https://steamcommunity.com/sharedfiles/filedetails/?id=3397284014

Original Author: 欧影Shiomi

Maintainer: InuiDame

## Adds three new spells  
```
    *Starlight 
    Deals two different shapes of damage based on the magic type  
    *Chain Lightning 
    Deals chained damage  
    *Blowback  
    Knocks back targets in an area  
``` 
The corresponding spellbooks can be purchased in the magic shop during the next merchant restock after adding the mod.
  
## Adjustable options in config  
```
    *Refresh amount  
    *Magic price multiplier  
    *Magic power multiplier  
    *Magic range multiplier  
    *Mana cost multiplier  
```
## cfg
    `Elin\BepInEx\config\ExtraSpell.cfg`
